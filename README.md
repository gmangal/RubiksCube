Colors:
    The following are the numbers within the 3D array in the program that
    correspond to colors on the Rubik's cube. This information is in the 
    documentation of the program itself, but is copied here as well:

    1 - White
    2 - Yellow
    3 - Red
    4 - Orange
    5 - Green
    6 - Blue
